Mostrar imagen de perfilâ€‹

  1. En el archivo imagen_perfil_widget.dart creamos un stateless widget llamado ImagenPerfil.

  2. Definimos un variable llamada url de tipo String.

  3. Dentro del mÃ©todo build retornamos un Container con padding de 20 a cada lado, un Center para centrar la imagen, un CircleAvatar para mostrar la imagen de forma redonda y un NetworkImage para obtener una imagen desde internet.

import 'package:flutter/material.dart';

class ImagenPerfil extends StatelessWidget {

  final String url;

  ImagenPerfil({
    required this.url
  });

  @override
  Widget build(BuildContext context) {
    return Container(
      padding: EdgeInsets.all(20),
      child: Center(
        child: CircleAvatar(
          radius: 100,
          backgroundImage: NetworkImage(url),
        ),
      ),
    );
  }
}

Mostrar menu de opcionesâ€‹

  1. En el archivo opcion_menu_widget.dart agregar un stateless widget llamado OpcionMenu.

  2. Declaramos 2 variables icon de tipo Icon (es un widget) y text de tipo String.

  3. El mÃ©todo build retorna un Container > Row > Icon, SizedBox, Text

import 'package:flutter/material.dart';

class OpcionMenu extends StatelessWidget {

  OpcionMenu(this.icon, this.text);

  final Icon icon;
  final String text;

  @override
  Widget build(BuildContext context) {
    return Container(
      padding: EdgeInsets.all(20), // padding a cada lado
      child: Row(
        children: [
          icon,

          SizedBox(width: 20,), // espacio entre icono y texto

          Text(
            text,
            style: TextStyle(color: Colors.black54, fontSize: 20),),
        ],
      ),
    );
  }
}
